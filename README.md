# JavaScript Core Concepts Assignment

## 📌 Description
In this project, I demonstrate my understanding of core JavaScript concepts by working with conditionals, switch statements, and ternary operators, handling type conversion and truthy or falsy values, creating functions with logical operators, building objects with methods, and using loops and the spread operator to solve problems.

## 🧠 Concepts Covered
- Control flow (if, switch, ternary)
- Type conversion
- Truthy and falsy values
- Functions and logical operators
- Objects and methods
- Arrays and spread operator
- Loops (for and while)

## 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Shaquan-Legae/JavaScript-Fundamentals.git
   ```
2. Navigate into the folder:
   ```bash
   cd JavaScript-Fundamentals
   ```
3. Run the file using Node.js:
   ```bash
   node index.js
   ```

## 📊 Features Implemented
- Grade calculator using switch statement
- Pass/fail checker using ternary operator
- Type conversion and truthy/falsy evaluation
- Custom greeting function
- Social media post object with methods
- Array merging using spread operator
- Pattern printing using loops
